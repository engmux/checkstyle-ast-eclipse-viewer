Welcome to Checkstyle AST Eclipse Viewer

Eclipse Checkstyle AST-Viewer plugin is a simple tool for displaying Checkstyle AST tree on Java files are currently opened in Eclipse. Just install it and use by clicking "Open Checkstyle AST Tree" context menu item on any Java file.

This project is maintained by sevntu-checkstyle
